## 二、服务器环境部署
### 2.1 连接服务器
使用SSH工具(Xshell/Putty)连接VPS，推荐设置密钥登录增强安全性

### 2.2 宝塔面板安装
bash
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

安装完成后记录面板地址、用户名及随机初始密码

1. 安装LNMP环境（Nginx 1.20+MySQL 5.7+PHP 7.4）

## 四、WordPress安装流程
1. 下载最新中文版安装包

2. 解压至网站根目录
3. 创建MySQL数据库并授权
4. 通过浏览器完成安装向导
